## 실행 방법
> 직접 실행해보고 싶다면?
1. 우선 [SilverEscape.zip](./SilverEscape.zip) 파일을 다운로드합니다. 
2. 위의 링크로 가서 우측의 `Download` 버튼을 누릅니다.
3. 다운로드 후 압축을 풀고 `MyTest.exe` 파일을 실행합니다.
